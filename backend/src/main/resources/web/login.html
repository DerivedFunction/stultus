Stultus - Social Media Platform

Welcome to Stultus! Stultus is a social media platform where you can connect with others, share posts, and engage in discussions. It provides authentication and authorization features, allows users to like or vote on posts, comment on posts, and manage their user profiles.

Features:

  • Authentication and authorization
  • Voting mechanism for posts
  • Text-based commenting system
  • User profile management
  • Post creation and editing
